## 輸入
第一行包含兩個整數 $n$、$m$。  
第一行包含兩個整數 $N$、$M$。  
接下來有 $q$ 行。  
接下來有 $Q$ 行。  
第二行包含 $N$ 個數字 $a_1, a_2, \cdots, a_N$。  
第二行包含 $N$ 個數字 $A_1, A_2, \cdots, A_N$。  
第二行包含 $n$ 個數字 $a_1, a_2, \cdots, A_n$。  
第二行包含 $n$ 個數字 $A_1, A_2, \cdots, A_n$。  

## 輸出
輸出運算結果。注意，老師要求好學生書寫算式要過程完整，字型整齊。因為小明是個好學生，所以他被要求按照嚴格的格式書寫！要把原本的題目謄寫一遍再加上答案喔。  

## 輸入限制
 - $1 \leq n \leq 10^4$
 - $1 \leq N \leq 10^4$
 - $1 \leq m \leq 10^4$
 - $1 \leq M \leq 10^4$
 - $1 \leq k \leq 10^4$
 - $1 \leq K \leq 10^4$
 - $1 \leq q \leq 10^4$
 - $1 \leq Q \leq 10^4$
